# Hero Video Instructions

Place a video file named `hero.mp4` in this folder.

## Recommended specs

- **Duration:** 6–15 seconds (loops seamlessly)
- **Resolution:** 1920×1080 minimum
- **Format:** MP4 (H.264) — compress to under 8 MB
- **Content ideas:** chef hands rolling sushi, sake being poured, curricanes close-up, restaurant ambiance
- **No audio needed** (will be muted)

Free tool to compress: [handbrake.fr](https://handbrake.fr)
